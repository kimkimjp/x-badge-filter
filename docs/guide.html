X Badge Filter

導入ガイド - 初心者向けインストール手順 v2.5.0

X（旧Twitter）のタイムラインから、フォローしていない認証バッジ付きアカウントの投稿を自動で非表示にするツールです。

機能説明
バッジフィルタ認証バッジ付き非フォローアカウントを自動非表示
ゼロフラッシュCSS Pre-hidingにより、バッジ投稿が一瞬も表示されずにフィルタ
ホワイトリスト特定アカウントをフィルタから除外
非表示バー「表示」「常に表示」ボタン付き
カウンター非表示にした件数を表示
マルチプラットフォームPC / Android / iPhone 対応

フィルタリングの流れ（v2.5.0）

→

→
バッジ確認
→
フォロー確認
→
表示 or 非表示

方法A: Chrome / Edge にインストール

Step 1. フォルダをダウンロード

Gitが使える場合はターミナルで以下を実行：

git clone https://github.com/kimkimjp/x-badge-filter.git

Gitがわからない場合は、「Code」ボタン → 「Download ZIP」 → 展開

Step 2. 拡張機能ページを開く

アドレスバーに以下を入力してEnter：

chrome://extensions/

Edgeの場合は edge://extensions/

Step 3. 「デベロッパーモード」をONにする
拡張機能 デベロッパーモード　● ← ここをON
Step 4. 「パッケージ化されていない拡張機能を読み込む」をクリック
パッケージ化されていない拡張機能を読み込む ← このボタンをクリック
Step 5. 「x-badge-filter」フォルダを選択

ファイル選択画面で、ダウンロードした「x-badge-filter」フォルダを選んで「フォルダーの選択」をクリック

Step 6. 完了！

ブラウザ右上のツールバーに「XBF」アイコンが表示されます。クリックすると設定画面が開きます。

方法C: Androidスマホにインストール（Firefox推奨）

Step 1. Firefox をインストール

Google Playストアで「Firefox」を検索してインストール（無料）

Step 2. Tampermonkey アドオンを追加

Firefoxを開く → 右下「⋮」メニュー → 「アドオン」→ Tampermonkeyの「＋」ボタンをタップ

「Tampermonkey」が一覧にない場合は、「アドオンをもっと探す」から検索してください。
Step 3. スクリプトをインストール

Firefoxで以下のURLを開くと、Tampermonkeyがインストールを確認します：

https://github.com/kimkimjp/x-badge-filter/raw/master/userscript/x-badge-filter.user.js

「インストール」をタップして完了。

Step 4. 完了！

方法D: iPhone / iPad にインストール

Step 1. Userscripts アプリをインストール

App Storeで「Userscripts」（開発者: Justin Wasack）を検索してインストール（無料）

Step 2. Safari拡張機能を有効にする
設定 → Safari → 拡張機能 Userscripts　● ← ONにする
Step 3. スクリプトファイルを保存

userscript/x-badge-filter.user.js をiPhoneにダウンロードし、Userscriptsアプリのスクリプトフォルダに保存します

Step 4. 完了！

Safariでx.comを開くと自動で動作します。画面右下に「XBF」ボタンが表示されます。

iOSのSafariでは一部の機能（API傍受）が制限される場合があります。その場合はDOM解析モードで動作します。

タイムラインでの表示

ボタン動作
「表示」その投稿を一時的に表示します（ページを離れるとまた非表示）
「常に表示」そのアカウントをホワイトリストに追加し、今後は常に表示

ゼロフラッシュ技術（v2.5.0新機能）

従来： ツイート表示 → フィルタ判定 → 非表示（一瞬見える）
v2.5.0： CSS Pre-hide → ツイート検出（見えない状態） → 判定 → 表示 or 非表示

バッジ付き投稿がそもそも画面に表示されることなく、スクロールしても視界に入りません。

よくある質問

Q. フォロー中のアカウントも非表示になる？

A. いいえ。フォローしているアカウントの投稿は表示されます。「フォローしていない」かつ「バッジ付き」のアカウントのみが対象です。

Q. 非表示にしたくないバッジ付きアカウントがある場合は？

A. 「常に表示」ボタンをクリックするか、設定画面のホワイトリストに @ユーザー名 を追加してください。

Q. 広告も非表示になる？

A. いいえ。広告（プロモーション）には影響しません。

Q. XのアカウントがBANされたりする？

A. このツールはブラウザの画面表示を変えているだけで、Xのサーバーには何も送信しません。アカウントに影響はありません。

Q. 設定はどこに保存される？

A. お使いの端末内にのみ保存されます。外部には一切送信されません。

Q. 動かない場合は？

A. ① x.comを開いているか確認 ② 拡張機能/スクリプトがONか確認 ③ ページを再読み込み ④ 「おすすめ」タブか確認（「フォロー中」タブでは効果なし）

# X Badge Filter

**X（旧Twitter）のタイムラインから、フォローしていない認証バッジ付きアカウントの投稿を自動で非表示にするツールです。**

「おすすめ」タイムラインに表示される、フォローしていないのに流れてくるバッジ付きアカウントの投稿を自動的にフィルタリングします。v2.5.0ではCSS Pre-hiding技術により、バッジ付きアカウントの投稿がそもそも画面に表示されることなくフィルタリングされます。

---

## どんなことができる？

- フォローしていない **認証バッジ付き** アカウントの投稿を非表示
- 特定のアカウントだけ表示する **ホワイトリスト** 機能
- 非表示にした投稿をワンタップで **一時表示** できる
- 非表示にした件数を **カウント表示**
- **ゼロフラッシュ**: バッジ付き投稿が一瞬も表示されずにフィルタリング

## インストール方法

### 方法A：PC（Chrome / Edge）にインストールする

パソコンのChromeまたはEdgeブラウザに拡張機能として入れる方法です。

#### 手順

**1. このフォルダをパソコンにダウンロードする**

Gitが使える場合：
```
git clone https://github.com/kimkimjp/x-badge-filter.git
```

Gitがわからない場合：
- このページの「Code」ボタン →「Download ZIP」をクリック
- ダウンロードしたZIPファイルを展開（右クリック → すべて展開）
- 展開した「x-badge-filter」フォルダの場所を覚えておく

**2. Chromeの拡張機能ページを開く**

- Chromeのアドレスバーに `chrome://extensions/` と入力してEnter
- （Edgeの場合は `edge://extensions/`）

**3. 「デベロッパーモード」をONにする**

- 画面右上にある「デベロッパーモード」のスイッチをクリックしてONにする

```
                                    ここをクリック
                                         ↓
┌──────────────────────────────────────────────┐
│  拡張機能                    [デベロッパーモード ○→●]  │
└──────────────────────────────────────────────┘
```

**4. 「パッケージ化されていない拡張機能を読み込む」をクリック**

- 画面左上に表示されるボタンをクリック

```
┌──────────────────────────────────────────────┐
│ [パッケージ化されていない拡張機能を読み込む]          │
│ [拡張機能をパック]                                │
└──────────────────────────────────────────────┘
```

**5. フォルダを選択**

- ファイル選択画面が開くので、先ほどの「x-badge-filter」フォルダを選んで「フォルダーの選択」をクリック

**6. 完了！**

- 拡張機能の一覧に「X Badge Filter」が表示されればOK
- ブラウザの右上にアイコンが追加されます

```
  ブラウザのツールバー
┌─────────────────────────────┐
│  ← → ↻  x.com          [XBF]  │  ← このアイコンが追加される
└─────────────────────────────┘
```

### 方法B：PC（Firefox）にインストールする

**1. フォルダをダウンロード**（方法Aの手順1と同じ）

**2. Firefoxの一時アドオンページを開く**

- アドレスバーに `about:debugging#/runtime/this-firefox` と入力してEnter

**3. 「一時的なアドオンを読み込む」をクリック**

**4. `manifest.json` ファイルを選択**

- 「x-badge-filter」フォルダの中にある `manifest.json` をクリックして「開く」

**5. 完了！**

> **注意**: Firefoxの一時アドオンはブラウザを閉じると消えます。
> 毎回読み込み直すのが面倒な場合は、Firefox Developer Edition を使うと
> 永続的にインストールできます。

### 方法C：Androidスマホにインストールする

Androidでは「ユーザースクリプト」という仕組みを使います。
**Firefox**（推奨）または **Kiwi Browser** のどちらかを使います。

#### 方法C-1：Firefox（推奨）

Firefoxは公式にアドオン（拡張機能）をサポートしており、最も安定して動作します。

**1. Firefox をインストール**

- Google Playストアで「Firefox」を検索してインストール（無料）

**2. Tampermonkey アドオンを追加**

- Firefoxを開く
- 右下の「⋮」メニュー → 「アドオン」をタップ
- 「アドオンマネージャー」画面で「Tampermonkey」の横にある「＋」ボタンをタップ
- 「追加」をタップして確認

```
┌──────────────────────────┐
│  おすすめのアドオン        │
├──────────────────────────┤
│  Tampermonkey        [＋] │  ← ここをタップ
│  uBlock Origin       [＋] │
│  Dark Reader         [＋] │
└──────────────────────────┘
```

> **ヒント**: 「Tampermonkey」が一覧にない場合は、右上の「アドオンをもっと探す」からFirefoxアドオンサイトで検索してインストールできます。

**3. ユーザースクリプトをインストール**

- Firefoxで以下のURLを開く：
  - `https://github.com/kimkimjp/x-badge-filter/raw/master/userscript/x-badge-filter.user.js`
- Tampermonkeyが「新しいスクリプトをインストールしますか？」と聞いてくるので「インストール」をタップ

**4. 完了！**

- FirefoxでX（x.com）を開くと自動で動作します
- 画面右下に「XBF」ボタンが表示されます（設定はここから）

```
┌─────────────────────┐
│                     │
│   タイムライン       │
│                     │
│           [XBF]     │  ← この丸いボタンが表示される
│                     │
│              [Post] │  ← X.comのツイートボタン
└─────────────────────┘
```

#### 方法C-2：Kiwi Browser（代替手段）

> **注意**: Kiwi Browserは2025年1月に開発終了（アーカイブ済み）となりました。
> 現在もインストール・使用は可能ですが、今後セキュリティアップデートは提供されません。
> 新規の方にはFirefox（方法C-1）を推奨します。

**1. Kiwi Browser をインストール**

- Google Playストアで「Kiwi Browser」を検索してインストール

**2. Kiwi Browser で Tampermonkey を追加**

- Kiwi Browserを開く
- 右上の「⋮」メニュー → 「拡張機能」（Extensions）をタップ
- 「+ (from store)」をタップしてChrome Web Storeが開く
- 「Tampermonkey」を検索して「Chromeに追加」をタップ

> **うまくいかない場合**: Tampermonkey v5.2以降でKiwi Browserとの互換性問題が報告されています。
> その場合は「Violentmonkey」を代わりにインストールしてみてください。

**3. ユーザースクリプトをインストール**

- Kiwi Browserで以下のURLを開く：
  - `https://github.com/kimkimjp/x-badge-filter/raw/master/userscript/x-badge-filter.user.js`
- Tampermonkey/Violentmonkeyが「インストールしますか？」と聞いてくるので「インストール」をタップ

### 方法D：iPhone / iPad にインストールする

iPhoneでは「Userscripts」というアプリを使います。

#### 必要なもの

- **Userscripts**（App Storeから無料でインストール）

#### 手順

**1. Userscripts アプリをインストール**

- App Storeで「Userscripts」（開発者: Justin Wasack）を検索してインストール

**2. Safari拡張機能を有効にする**

- 「設定」アプリ → 「Safari」→ 「拡張機能」
- 「Userscripts」をONにする

```
  設定 → Safari → 拡張機能
┌──────────────────────────┐
│  Userscripts     [○→●]    │  ← ONにする
└──────────────────────────┘
```

**3. スクリプトファイルを保存**

- このリポジトリの `userscript/x-badge-filter.user.js` をiPhoneにダウンロード
- 「ファイル」アプリに保存する
  （iCloud DriveやiPhone本体のどこでもOK）

**4. Userscriptsアプリにスクリプトを登録**

- Userscriptsアプリを開く
- スクリプトの保存先フォルダを設定（初回のみ）
- 先ほどの `x-badge-filter.user.js` をそのフォルダに移動またはコピー

**5. 完了！**

- SafariでX（x.com）を開くと自動で動作します
- 画面右下に「XBF」ボタンが表示されます

> **注意**: iOSのSafariでは一部の機能（API傍受）が制限される場合があります。
> その場合はDOM解析モードで動作します。

### タイムラインでの動作

フィルタリングされた投稿は、以下のようなバーに置き換わります：

```
┌──────────────────────────────────────────┐
│ @example の投稿を非表示にしました  [表示] [常に表示] │
└──────────────────────────────────────────┘
```

- **「表示」** → その投稿を一時的に表示します（ページを離れるとまた非表示に）
- **「常に表示」** → そのアカウントをホワイトリストに追加し、今後は常に表示します

## 技術的な特徴（v2.5.0）

### ゼロフラッシュ フィルタリング

従来のフィルタ拡張機能では、ツイートが一瞬表示されてから非表示になる「フラッシュ」が発生していました。v2.5.0では **CSS Pre-hiding** 技術を導入し、この問題を解消しています。

```
従来の方式:
  DOM追加 → 画面に表示 → フィルタ判定 → 非表示（フラッシュ発生）

v2.5.0:
  CSS Pre-hiding → DOM追加（見えない状態） → フィルタ判定 → 表示 or 非表示
```

- ページ読み込み時にCSS規則を即座に注入し、未処理のツイートを非表示状態にする
- MutationObserverで同期的にバッジ有無を判定
- バッジなしツイートは即座に表示、バッジありツイートはフォロー判定後に表示/非表示を決定
- 安全弁として3秒のタイムアウトを設定（処理できなかったセルは自動表示）

### フォロー判定の仕組み

| 方式 | 説明 | 対応環境 |
|------|------|---------|
| API傍受 | X.comのAPIレスポンスからフォロー状態を取得 | PC（Chrome/Edge/Firefox） |
| DOM解析 | ツイート内の「フォロー」ボタンのテキストから判定 | 全環境（Android/iPhone含む） |

PC版ではAPI傍受を優先し、より正確なフォロー判定を行います。Android Firefox + Tampermonkey環境ではDOM解析モードで動作します。

## アンインストール方法

### Chrome / Edge
1. `chrome://extensions/`（Edgeは `edge://extensions/`）を開く
2. 「X Badge Filter」の「削除」ボタンをクリック

### Firefox
1. `about:addons` を開く
2. 「X Badge Filter」の「...」→「削除」

### Android（Tampermonkey）
1. Tampermonkeyのダッシュボードを開く
2. 「X Badge Filter」のゴミ箱アイコンをタップ

### iPhone（Userscripts）
1. Userscriptsアプリのフォルダから `x-badge-filter.user.js` を削除
2. または「設定」→「Safari」→「拡張機能」→ Userscripts をOFFにする

## トラブルシューティング

### 動かない場合

1. **X（x.com）を開いているか確認** -- twitter.com でも動作します
2. **拡張機能がONか確認** -- 設定画面のトグルがONになっているか
3. **ページをリロード** -- Ctrl+R（Mac: Cmd+R）で再読み込み
4. **「おすすめ」タブか確認** -- 「フォロー中」タブでは元からフォローしていないアカウントは表示されないため、フィルタの効果がありません

### スマホで「XBF」ボタンが表示されない

1. ユーザースクリプトマネージャー（Tampermonkey等）が有効か確認
2. スクリプトがONか確認（Tampermonkeyのダッシュボードで確認）
3. ページを再読み込み
